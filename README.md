# GreenGo - Brisbane Bike Path Explorer

## Project Overview

GreenGo is a comprehensive web application designed to enhance the cycling experience in Brisbane. It provides cyclists with tools to explore bike paths, discover events, share reviews, and connect with the cycling community. Our mission is to promote eco-friendly transportation and encourage a healthier lifestyle through cycling.

## Features
1. **Interactive Bike Path Map**: Utilizes Leaflet.js to display an interactive map of Brisbane's bike paths.
2. **Path Finder**: Allows users to filter bike paths and drinking fountains based on various criteria.
3. **Event Discovery**: Lists cycling events in Brisbane with details and map locations.
4. **User Reviews**: Enables users to share their experiences and read others' reviews.
5. **Google OAuth Integration**: Secure user authentication using Google accounts.
6. **User Profiles**: Personalized profiles for users to track their cycling statistics and achievements.
7. **Responsive Design**: Ensures a seamless experience across desktop and mobile devices.


## Coding Conventions

- **Indentation**: 4 spaces for all file types (PHP, HTML, CSS, JavaScript)
- **Naming Conventions**:
  - Variables and functions: camelCase (e.g., `getUserProfile()`)
  - CSS classes: kebab-case (e.g., `bike-path-info`)
  - File names: lowercase with hyphens (e.g., `find-path.js`)
- **Comments**: Inline comments for complex logic
- **CSS**: Use of CSS variables for consistent theming
- **JavaScript**: ES6+ features used where appropriate
